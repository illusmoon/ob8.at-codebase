Flattr
------------------------

To install, place the entire flattr folder into your modules directory.
Go to Administer -> Site building -> Modules and enable the flattr module.

Administration
------------------------
Go to Admin -> Content -> Types and edit a existing content type. There you can enable the flattr button
by setting a default category in the flattr fieldset.

User
------------------------
If you edit your account you set your flattr uid. 

Thats it.
